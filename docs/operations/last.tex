\subsubsection{Last}
The operation $last$ will return the last element of a
\begin{itemize}
\item String Scalar
\item Array Scalar
\item Collection
\end{itemize}
such that
$$last(arg) = arg_{j}$$
and when $arg$ is a Array Scalar or Collection
$$last(X) \implies last(<x_{i}..x_{n}..x_{j}>) = x_{j}$$
and when $arg$ is a String Scalar
$$last(abc) \implies last(<a, b, c>) = c$$
